# タスク01: 依存パッケージのインストール

## 概要
プロジェクトに必要な依存パッケージをインストールします。

## インストールするパッケージ

### UI/UXライブラリ
- `lucide-react` - アイコンライブラリ
- `framer-motion` - アニメーションライブラリ

### 状態管理
- `zustand` - 軽量な状態管理ライブラリ

### ユーティリティ
- `date-fns` - 日付操作ライブラリ

## コマンド
```bash
npm install lucide-react zustand date-fns framer-motion
```

## 確認方法
- `package.json` に上記パッケージが追加されていることを確認
- `node_modules` に各パッケージがインストールされていることを確認
