# タスク07: タスクコンポーネントの実装

## 概要
タスク表示と操作に関する主要コンポーネントを実装します。

## ファイル構成
```
components/tasks/
├── TaskCard.tsx      # 個別のタスクカード
├── TaskList.tsx      # タスクリスト全体
└── TaskModal.tsx     # タスク追加/編集モーダル
```

## TaskCard コンポーネント

### 表示内容
- タスクタイトル
- 説明文（あれば）
- 期限日時（`date-fns` でフォーマット）
- 優先度バッジ（Badge コンポーネント）
- 完了チェックボックス
- 編集・削除ボタン

### アクション
- チェックボックスで完了状態をトグル
- 編集ボタンでモーダルを開く
- 削除ボタンでタスクを削除（確認ダイアログ）

### デザイン
- shadcn/ui の Card コンポーネントを使用
- 優先度に応じた色分け（low: 緑、medium: 黄、high: 赤）

## TaskList コンポーネント

### 機能
- タスクを一覧表示
- 完了/未完了でフィルタリング
- 期限日時でソート

### レイアウト
- グリッドまたはリストレイアウト
- レスポンシブ対応

## TaskModal コンポーネント

### 機能
- 新規タスク追加
- 既存タスク編集

### フォーム項目
1. タイトル（必須）
2. 説明（オプション）
3. 期限日時（DatePicker）
4. リマインド時間（Select: 0, 5, 15, 60分前）
5. 優先度（Select: low, medium, high）

### バリデーション
- タイトルは必須
- 期限は未来の日時のみ

### 送信処理
- 新規の場合: `addTask` を呼び出し
- 編集の場合: `updateTask` を呼び出し
- モーダルを閉じる

## 注意事項
- フォームの状態管理（React Hook Form や useStateで管理）
- Date型の入力は HTML の `datetime-local` または日付ピッカーライブラリを使用
