# タスク06: レイアウトコンポーネントの実装

## 概要
アプリケーション全体のレイアウトとヘッダーコンポーネントを実装します。

## ファイル
- `components/layout/Header.tsx`

## Header コンポーネント

### 機能
1. アプリケーションタイトル表示
2. ユーザーの表示名を表示
3. 新規タスク追加ボタン

### デザイン要件
- 清潔感のあるモダンデザイン
- レスポンシブ対応（モバイル/デスクトップ）
- Tailwind CSSでスタイリング

### レイアウト例
```
+--------------------------------------------------+
| 📋 タスクリマインダー      [ユーザー名] [+ 新規] |
+--------------------------------------------------+
```

## App Router での統合
- `app/layout.tsx` を更新してHeaderを組み込む
- クライアントコンポーネントとして実装（`"use client"` ディレクティブ）

## 注意事項
- ヘッダーは固定位置（sticky header）にするか検討
- モバイルビューでは適切に折り返し
